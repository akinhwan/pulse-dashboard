## Quick start

- Clone the repo: `git clone https://github.com/akinhwan/pulse-dashboard.git`.
    - `npm install`
    - `npm run serve`
- [Download from Github](https://github.com/akinhwan/pulse-dashboard.git/archive/master.zip).

## Reporting Issues

We use GitHub Issues as the official bug tracker for the Pulse Dashboard. Here are some advices for our users that want to report an issue:

1. Make sure that you are using the latest version of the Pulse Dashboard.
2. Providing us reproducible steps for the issue will shorten the time it takes for it to be fixed.
3. Some issues may be browser specific, so specifying in what browser you encountered the issue might help.
